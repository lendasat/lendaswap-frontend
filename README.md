# Lendaswap Frontend

Frontend monorepo for Lendaswap - atomic swaps between Bitcoin (via Lightning or Arkade) and Polygon tokens.

Built with [turbo-repo](https://turborepo.com/) for monorepo management and React for the UI.

## Applications

The main application is located in `apps/lendaswap/`.

## Getting Started

```sh
pnpm run dev
```

To create a production bundle:

```sh
pnpm run build
```
